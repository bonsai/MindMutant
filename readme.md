# MindMutant
**Evolutionary Text Synthesis Engine**

> **"From Chaos to Meaning"**
> LLMに依存せず、進化の力で言葉を紡ぐ。

MindMutantは、遺伝的アルゴリズム（Genetic Algorithms）とGPUコンピューティングを融合させた、実験的なテキスト生成エンジンです。
巨大な言語モデルによる確率的な予測ではなく、生物学的進化を模倣した「淘汰」と「変異」のプロセスを通じて、無秩序な単語のスープから意味のあるアイデアを創発させます。

## Core Philosophy

**デタラメから、意味のあるものを残す。**

私たちの脳内で行われている創造のプロセス――無数の断片的な情報の結合と、意識による選別――を、アルゴリズムとして再現します。

*   **Non-LLM Genesis**: ブラックボックス化されたモデルではなく、透明性の高い進化的アプローチを採用。
*   **Symbolic Evolution**: 言葉のアイデア、画像生成プロンプト、楽曲のコード進行。単純なシンボル操作から、複雑な構造を持つ「概念」へと進化させます。
*   **Human-in-the-Loop**: アルゴリズムによる適応度評価に加え、ユーザー自身の美的感覚を「淘汰圧」としてシステムに注入。あなたの選択が、次の世代の進化を決定づけます。

## Architecture

1.  **Population Initialization**: ランダムな単語とシンボルの集合体を生成。
2.  **Evaluation (Fitness)**: 構文解析と意味的評価による自動フィルタリング。
3.  **Selection & Mutation**: 優秀な個体を交配・変異させ、次世代を生成。ここでColab GPUのパワーを活用し、高速な世代交代を実現します。
4.  **User Feedback**: ユーザーによる直接的な介入により、進化の方向性を操舵します。

## Requirements

*   Google Colab (GPU Recommended)
*   Python 3.8+
*   FastAPI / Uvicorn (Viewer)
